# Rocketshoes

It is a project created on study ReactJs + Redux and Redux Saga.
The idea is building a interface similar an cart of ecommerce.

In this project i was used:

- React
- React Router Dom
- Reactotron
- Redux
- React Redux
- Redux Saga
- Immer
- Axios
- Polished
- Styled Components
- Eslint
- Prettier
- EditorConfig
